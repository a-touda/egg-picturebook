# ユースケース

### v1での実装範囲
取り合えず基本的な機能が動くようにする

- タイトル画面
  - 「たまごをつくる」のボタン（カスタマイズ画面にとぶ）

- カスタマイズ画面
  - カスタマイズするたまご（左側）
  - 大きさを設定するラジオボタン（仮）
  - たまごのベースの色を設定する（カラーピッカー）
  - たまごの模様を設定するボタン（縞・ドット）
  - 決定ボタン（結果画面にとぶ）
- RGBの値からベースの色を12色に判別する機能
  - 赤・桃・紫・青・水色・緑・黄緑・黄・橙・茶・黒・白の12色
- 模様の有無を判定する機能
  - 縞：1種、ドット：１種　の計２種
- いきものの種類
  - ベースのいきもの：１２種
  - 派生：各２種
  - 12 × 3 = 36   【全36種】
 
- 結果画面
  - 中央にカスタマイズしたたまご
  - たまごが割れるようなアニメーション（光がたまごの中心からあふれる感じ）
  - 中央に生まれたいきもの
  - 「タイトルに戻る」のボタン（タイトル画面にとぶ）

- たまごが持つ要素
  - 大きさ
  - ベースの色
  - 模様
  
### v2での実装範囲
時間があったら追加する感じで

- 図鑑画面
  - うまれたいきものの一覧を表示
  - 画像をクリックorタップで拡大表示する機能
  - 「タイトルに戻る」ボタン
- 「ずかんを見る」のボタン（タイトル画面」と結果画面に設置）
- 結果画面に新しいいきものだった場合メッセージを表示する機能
